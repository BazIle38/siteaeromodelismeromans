# Club d'aéromodélisme Romans

Site vitrine du **Club d'aéromodélisme Romans** — association de passionnés de modèles réduits volants à Romans-sur-Isère et ses environs.

## Fonctionnalités

- Présentation du club et de ses activités
- Carte interactive du terrain de vol (Leaflet + OpenStreetMap)
- Coordonnées et formulaire de contact
- Design responsive et animations CSS
